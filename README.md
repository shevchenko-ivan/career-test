# Знайди свій шлях — Тест профорієнтації

Інтерактивний тест профорієнтації для підлітків (14-18 років), який допомагає визначити інтереси, цінності та сильні сторони для вибору майбутнього напрямку.

## Що всередині

- **36 питань** у 4 розділах: інтереси (парний вибір), flow-активності, цінності (сценарії) та навички.
- **Модель RIASEC** Джона Голланда — сучасний стандарт профорієнтації (понад 70 років досліджень).
- **Понад 100 професій** — включно з сучасними та нішевими напрямками (наприклад, AI-етик, UX-письменник, спеціаліст з climate tech тощо).
- **Пошук університетів** — огляд можливостей безкоштовного навчання для українців у Європі (Німеччина, Іспанія, Норвегія та ін.) та зручний пошук.
- **Питання для обговорення** — блок питань для саморефлексії або розмови з друзями та близькими після проходження тесту.

## Як користуватися

Просто відкрий файл `index.html` у будь-якому браузері на комп'ютері або телефоні.

## Технології

- Один HTML-файл, без зовнішніх залежностей.
- Vanilla JS та класичний CSS.
- Повністю працює офлайн (результати розраховуються локально).
