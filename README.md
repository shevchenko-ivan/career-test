# Знайди свій шлях — Тест профорієнтації

Інтерактивний тест профорієнтації для підлітків (14-18 років), який допомагає визначити інтереси, цінності та сильні сторони для вибору майбутнього напрямку.

## Що всередині

- **36 питань** у 4 розділах: інтереси (парний вибір), flow-активності, цінності (сценарії), навички
- **Модель RIASEC** Джона Голланда — золотий стандарт профорієнтації, 70+ років досліджень
- **100+ професій** включно з нішевими та сучасними напрямками (AI-етик, UX-письменник, climate tech тощо)
- **Пошук університетів** в Європі з безкоштовним навчанням для українців (Німеччина, Іспанія, Норвегія та ін.)
- **Питання для обговорення** — щоб поговорити з близькими після тесту
